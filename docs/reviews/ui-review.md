# UI Review — MiniExplorer

- **Date**: 2026-02-02
- **Scope**: Prototype (Explore / Companion)

## Issues
1. **导航返回按钮可触达性**：Back 按钮与边缘间距较小，建议增加 hit area。
2. **录音状态反馈一致性**：Explore/Companion 状态文案与表情不完全统一，需统一枚举。
3. **主按钮层级**：主按钮与底部控制条视觉层级不够突出，建议增加对比度/阴影。
4. **摄像头占位信息**：Companion 小窗占位表情符号提示不足，建议增加“预览中”文案。
5. **安全区域适配**：顶部安全区占位与标题间距略大，建议根据机型微调。

## Conclusions
- 现有原型可演示 Explore/Companion 的主流程，核心交互已具备。
- 需要在 iOS 实现时统一状态机与文案，避免双端不一致。
