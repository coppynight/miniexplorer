# MiniExplorer UI 原型设计文档

**日期:** 2025-01-25
**状态:** 已确认

---

## 设计理念

类似视频通话的实时对话体验，为 3-6 岁儿童设计，强调：
- 图标 + 简单词组合（非纯图标，避免歧义）
- 即时视觉反馈（动画、颜色变化）
- 触觉反馈（震动）
- 音效反馈

---

## AI 形象

**风格:** 圆润简约，渐变球体

**状态表情:**
- 😊 开心（默认）
- 😮 惊讶
- 🥰 喜欢
- 😴 休息

**动画:**
- 呼吸动画（上下轻微浮动）
- 说话动画（缩放 + 发光）
- 听动画（耳朵/触角微动）

## 探索模式界面

```
┌─────────────────────────────────────┐
│                                     │
├─────────────────────────────────────┤
│                                     │
│         ┌─────────────┐             │
│         │             │             │
│         │   相机预览   │             │  全屏后置相机
│         │   (全屏)    │             │
│         │             │             │
│         └─────────────┘             │
│                                     │
│              ╭──╮                   │  AI 小球状态指示
│              │😊│                   │  悬浮显示
│              ╰──╯                   │
│                                     │
├─────────────────────────────────────┤
│                                     │
│           ╭──────────╮              │
│           │  🎤 说话  │              │  图标 + 简单词
│           ╰──────────╯              │
│                                     │
│     [ 🔍 看看 ]    [ 😊 聊天 ]      │  底部 Tab
└─────────────────────────────────────┘
```

**按钮状态:**
| 状态 | 显示 | 颜色 |
|------|------|------|
| 空闲 | 🎤 说话 | 主色(蓝) |
| 对话中 | 🎵 听着呢 | 绿色+脉冲 |
| AI回复 | 😊 等等我 | 粉色 |
| 长按结束 | 👋 拜拜 | 灰色 |

**交互流程:**
1. 默认显示相机预览 + "说话"按钮
2. 点击按钮 → 自动拍照 → 建立实时语音连接
3. 对话中按钮变为"听着呢"，孩子自由说话
4. AI 回复时按钮变为"等等我"
5. 点击"拜拜"结束对话

| 用途 | 看外面世界 | 面对面聊天 |

## Tab 栏设计

**文案:**
- 探索模式: 🔍 看看

**状态:**
- 未选中: 灰色，正常大小
- 选中: 主色，略微放大

## 动画规格

| 组件 | 动画 | 时长 | 缓动 |
|------|------|------|------|
| 主按钮脉冲 | scale 1.0→1.05→1.0 | 1.5s | ease-in-out |
| AI 呼吸 | translateY 0→-5px→0 | 2s | ease-in-out |
| AI 说话 | scale 1.0→1.1→1.0 + glow | 0.3s | ease |
| Tab 切换 | bounce | 0.3s | spring |
| 拍照闪光 | opacity 0→1→0 | 0.2s | linear |

---

## 触觉反馈 (Haptic)

| 交互 | 震动类型 | iOS API |
|------|---------|---------|
| 点击开始对话 | 中等冲击 | `.impactMedium` |
| 结束对话 | 轻柔冲击 | `.impactLight` |
| 切换 Tab | 选择反馈 | `.selectionChanged` |
| 拍照 | 成功通知 | `.notificationSuccess` |
| AI 开始说话 | 轻柔冲击 | `.impactSoft` |
