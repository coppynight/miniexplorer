# MiniExplorer

儿童 AI 探索伙伴 - iOS 应用

## 简介

MiniExplorer 是一款面向 3-6 岁儿童的 AI 探索伙伴应用，通过实时语音和图像交互，用启发式对话陪伴孩子认识世界。

## 功能

- **探索模式**: 用后置摄像头拍东西，AI 介绍这是什么
- **陪伴模式**: 用前置摄像头，和 AI 语音聊天、陪玩

## 技术栈

- iOS 16+ / SwiftUI
- Coze WebSocket API (实时语音 + 图片)
- WKWebView + Coze JS SDK
